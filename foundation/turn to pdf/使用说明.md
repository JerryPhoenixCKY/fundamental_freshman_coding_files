# 文本转PDF工具 使用说明

## 功能介绍

这是一个Python GUI应用程序，可以：

1. **选项1：直接输入文本转PDF**
   - 复制从Gemini或其他来源获得的大段文字
   - 粘贴到输入框中
   - 程序会自动处理数学公式乱码
   - 生成格式完善的PDF文件

2. **选项2：Word文档转PDF**
   - 选择本地的.doc或.docx文件
   - 自动将其转换为可读的PDF文件
   - 保留文档中的文本和表格

## 安装步骤

### 第一步：安装依赖库

**方法1（推荐）：使用安装脚本**
- 双击运行 `安装依赖.bat` 文件
- 等待安装完成

**方法2：手动安装**
打开命令行（Win+R，输入cmd），复制以下命令：
```
pip install reportlab python-docx
```

### 第二步：运行程序

双击运行 `文本转PDF工具.py` 文件

或在命令行中执行：
```
python 文本转PDF工具.py
```

## 使用步骤

### 选项1：输入文本转PDF

1. 点击"选项1：输入文本生成PDF"按钮
2. 在新窗口中的文本框粘贴您的文本
3. 点击"生成PDF"按钮
4. 选择PDF保存位置
5. 完成！PDF已生成

### 选项2：转换Word文件

1. 点击"选项2：选择doc文件转为PDF"按钮
2. 在文件对话框中选择您的Word文件
3. 选择PDF保存位置
4. 完成！PDF已生成

## 常见问题

**Q: 为什么运行时显示缺少库？**
A: 请先运行"安装依赖.bat"文件安装必要的库。

**Q: 转换后的PDF显示乱码？**
A: 这通常是字体问题。程序会自动使用系统中的中文字体。如果没有找到，可能需要安装额外字体。

**Q: 可以转换.doc文件吗？**
A: 可以。程序支持.doc和.docx格式。

**Q: 如何从Gemini复制含有公式的文本？**
A: 直接在Gemini对话框中选择全文复制，粘贴到程序中即可。程序会自动清理乱码。

## 系统要求

- Windows 7 或更高版本
- Python 3.7 或更高版本
- 至少100MB硬盘空间

## 遇到问题？

如果程序无法正常运行，请检查：

1. Python是否正确安装
2. 依赖库是否已安装（运行安装脚本）
3. 输入文件格式是否正确
4. 文件路径是否包含特殊字符
